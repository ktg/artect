Instructions for creating documents

 

This is an example document, which can be modified to include your own content. It includes instructions for making these modifications, and some examples of the kinds of content and formatting that you can use in it. You will be able to view any modifications in the document browser window in ARTECT. Please ask if you would like to know more about how to do this.

 

1.     Instructions for loading/saving documents

 

In order to modify this document, you should load it up into Microsoft Word. To do this:

 

  1. Run Word
  2. Select the “Open” option from the “File” menu
  3. Select the C:\ drive in the drop-down box at the top of the file dialog
  4. Navigate down through directories named “inScape”, “ect”, “resources”, “arauthoring” and “documents”
  5. Open file custom1.html
  6. Make any changes that you want

 

You can also load up any of the other documents in this directory (named default.html, custom2.html, custom3.html … custom10.html).

 

Once you have finished making modifications, use the “Save as” option on the “File” menu to save them. You should select the “web page” option from the “Save as type” drop-down list, and make sure that you enter the file name custom1.html (not custom1.html as might be suggested by MS Word). Then click “Save”.

 

Please note that you should not use the “Save” option on the “File” menu as this will not save the file in the correct format for use in ARTECT.

 

2.     Types of document

 

There are two different types of document in the same directory as this one, both of which can be modified to include any features that you wish. Documents labelled custom1.html up to custom10.html can be attached to markers, and will be displayed in the document browser when any marker that they are attached to becomes visible in the video feed. The document labelled default.html will always be displayed when there are no markers in the video feed. You might use it to contain an introduction to your exhibit.

 

3.     Types of document content

 

You can include lots of different types of content in your document:

 

  1. Pictures (preferably GIF, JPG or PNG format – please note that pictures in BMP format will not work)
  2. Text (any size, any font, including bold, italic or underlined text)
  3. Tables
